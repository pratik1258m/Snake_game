# Neon Snake Game
A modern, cyberpunk-themed Snake game built with HTML5 Canvas and Vanilla JavaScript.

## Features
- **Neon Aesthetics**: Glowing visuals and dark mode design using CSS variables.
- **Smooth Gameplay**: Optimized game loop with `requestAnimationFrame`.
- **Responsive Design**: Works on desktop and mobile (touch controls included).
- **High Score**: Persists your best score using local storage.
- **Dynamic Speed**: The game gets faster as your score increases.
- **Particle Effects**: Visual explosions when collecting food.

## How to Play
- **Desktop**: Use Arrow keys to control the snake.
- **Mobile**: Swipe or use the on-screen D-pad.

## Setup
Simply open `index.html` in any modern web browser.
